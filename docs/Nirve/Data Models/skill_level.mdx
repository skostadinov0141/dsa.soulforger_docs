# SkillLevel

## Model

| Name | Type | Description |
| - | - | - |
| **level** | `int` | The relevant level of the skill |
| **cost** | `int` | The XP cost of the level. |
| **description** | `string` | The description of the skill. |
| **depends_on** | `srting[]` | In case the level requires you to aquire something else before that, the ObjectID of the thing it depends on is put here. |
