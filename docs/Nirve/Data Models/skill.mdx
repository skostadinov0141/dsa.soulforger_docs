# Skill

## Model

| Name | Type | Description |
| - | - | - |
| **id** | `int` | The unique identifier of the skill. The idintifier is based on the string representation of a MongoDB ObjectID. |
| **name** | `string` | The name of the skill. |
| **description** | `string` | The description of the skill. |
| **xp_cost** | `int` | The amount of experience points needed to unlock the skill. |
| **levels** | [`SkillLevel[]`](./skill_level.mdx) | The levels of the skill. |
